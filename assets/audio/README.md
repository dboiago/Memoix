# Timer Alarm Sounds

Place the following MP3 or WAV files in this folder:

1. `alarm_bell.mp3` - Classic bell/chime sound
2. `alarm_beep.mp3` - Digital beep sound  
3. `alarm_gentle.mp3` - Soft/gentle notification sound

## Recommended Sources for Free Sounds

- [Pixabay Sound Effects](https://pixabay.com/sound-effects/) - Free for commercial use
- [Freesound.org](https://freesound.org/) - Creative Commons sounds
- [Mixkit](https://mixkit.co/free-sound-effects/) - Free sound effects

## Requirements

- Format: MP3 or WAV
- Duration: 1-3 seconds recommended
- Keep file sizes small (< 100KB each)
